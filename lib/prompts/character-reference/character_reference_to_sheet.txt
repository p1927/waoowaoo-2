# Reference Image to Character Sheet Prompt (Image-to-Image Mode)

Based on provided reference image, extract character's facial features, hairstyle, body type and clothing style as reference.

## Art Style Preservation Rules

Maintain original image's art style and artistic style:
- Real photo style → Generate real photo style
- Anime style → Generate anime style
- Realistic illustration → Generate realistic illustration style
- Other styles → Maintain consistency

## Generation Rules

1. Ignore original image's specific color tone and lighting
2. Use natural soft studio lighting
3. Draw normal beautiful human body proportions
4. Don't copy original image's image quality, blur, noise or flaws
5. Generated image must be clear and sharp, rich in details, professional quality
6. Character expression should be natural calm neutral expression, eyes looking directly at camera

## Auto-Complete Missing Parts

If reference image is half-body or partial body, please reasonably complete unrevealed parts based on clothing style and character features:
- **Missing lower body**: Infer and draw matching pants/skirt based on upper garment style
- **Missing feet**: Add suitable shoe style based on overall outfit style
- **Missing hands/arms**: Reasonably complete based on posture
- Maintain overall style consistency, ensure completed parts coordinate and unify with visible parts
