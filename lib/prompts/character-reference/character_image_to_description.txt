# Image Reverse Character Description Prompt

Please analyze this character image and generate a detailed character appearance description (for AI image generation).

## Output Requirements

Generate a complete character visual description, containing following elements:

1. Gender and age range (e.g., approximately 25-year-old male)
2. Hairstyle and hair color (e.g., black short hair, slightly curly brown long hair)
3. Face shape and facial features (e.g., sword-like brows starry eyes, high nose bridge, thin lips)
4. Body type and figure (e.g., slender build, strong physique)
5. Clothing style (e.g., dark blue suit, white shirt, leather belt)
6. Accessory features (e.g., silver watch on left hand, golden brooch on chest)
7. Overall temperament keywords (e.g., elite temperament, ascetic style, aloof, warm gentle man)

## Missing Content Completion Rules

If reference image only shows part of body (e.g., upper body, headshot), please reasonably infer and complete based on existing information:
- **Missing lower body**: Infer pants/skirt type based on upper garment style (e.g., suit jacket with dark blue suit pants, casual top with jeans)
- **Missing shoes**: Infer shoe style based on overall outfit style (e.g., formal wear with leather shoes, casual wear with sneakers or canvas shoes)
- **Missing accessory details**: Reasonably add accessories based on character temperament (e.g., business style with watch, casual style with bracelet)

## Prohibited Descriptions

- Skin color
- Eye color
- Expression
- Action
- Background
- Pose

## Output Format

A coherent description text, approximately 200-300 characters, directly usable as image generation prompt.
Only return description text, don't have any titles, numbers or other formatting.
